# My Project

This is a simple calculator app built with React and TypeScript.

It can perform basic arithmetic operations like +, -, *, and รท.

# How to Run the Project

**1. Clone the repository**

`git clone https://github.com/PawanNeupan/demo-react-app.git`

**2. Go to the project folder**

`cd my-app`

**3. Install dependencies**

`npm install`

**4. Start development server**

`npm run dev`

**5. Open in Browser**

`http://localhost:5173`
